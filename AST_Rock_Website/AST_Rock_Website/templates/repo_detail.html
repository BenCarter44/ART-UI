Repositories By Link

This project automates the labeling of GitHub repositories based on their content, using a taxonomy of domains and subdomains. It implements a precision threshold of 75% to ensure that only high-confidence classifications are applied, enhancing the organization and searchability of repositories.

Your Respositores

Handles the core processing of data, including fetching and storing pull requests data. Creates Abstract Syntax Trees (AST) from the files in the pull requests, identifies APIs and functions, and establishes their connections. Uses AI to classify APIs into a set list of domains and functions into subdomains, storing these classifications in databases to minimize redundant processing.

Search by Link

Focuses on extracting and preparing data from GitHub repositories. Converts data into usable formats like CSV and pickle, making it easier for other teams to integrate this data into their systems. Cleans and fuses data from different sources to ensure it is accurate and comprehensive for further analysis.

The AST Team

Each team's efforts are interconnected, contributing to a comprehensive system that leverages machine learning and AI to enhance the management and categorization of software development tasks within GitHub repositories.
